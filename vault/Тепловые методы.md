# Тепловые методы увеличения нефтеотдачи

## Горячее заводнение

Горячее заводнение — это нагнетание в пласт горячей воды для снижения вязкости нефти и улучшения её подвижности.

### Принцип действия
- Снижение вязкости нефти при повышении температуры (экспоненциальная зависимость)
- Повышение давления в пласте
- Вытеснение нефти из пор с низкой адсорбционной ёмкостью

### Технические параметры
- Температура воды на входе: 40–80°C
- Расход: 10–50 м³/сут на скважину
- Применяется при вязкости нефти > 50 сП

### Эффективность
Прирост нефтеотдачи: 8–15% к начальным извлекаемым запасам при благоприятных геолого-физических условиях.

## Паротепловое воздействие

Впрыск пара в скважину для прогрева прибойного пространства.

### Параметры пара
- Давление: 20–40 атм
- Температура: 200–250°C
- Цикл закачки-вымывания: 3–6 месяцев

### Применение
Используется для скважин с вязкой нефтью (100–500 сП) на глубинах до 1500 м.
